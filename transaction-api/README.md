# transaction-api

This is an ASP.NET Core Web API project. You can run it locally for development and testing.

## Getting Started

1. **Restore dependencies:**
   ```sh
   dotnet restore
   ```
2. **Run the API:**
   ```sh
   dotnet run
   ```
3. The API will start on `http://localhost:5000` by default (no HTTPS).

## Project Structure
- `Program.cs`: Entry point and configuration.
- `appsettings.json`: App configuration.
- `Controllers/`: Add your API controllers here.
